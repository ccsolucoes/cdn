CC Soluções CDN Starter Package (Test Bundle)

This bundle simulates what you'd host under:
  https://cdn.ccsolucoes.com/

Included:
- cc-layout/1.0.0/cc-layout.css
- cc-reveal/1.0.0/cc-reveal.css
- cc-reveal/1.0.0/cc-reveal.js
- cc-reveal/stable/ (same files copied from 1.0.0)
- ccui/1.0.0/ccui-core.js
- ccui/1.0.0/ccui-slider.js
- ccui/1.0.0/ccui-motion.css
- ccui/stable/ (same files copied from 1.0.0)
- demos/index.html (open locally to test)

Local test:
Open demos/index.html in your browser. It references ../ paths so it works as a folder demo.

Production:
Upload the folders (cc-layout, cc-reveal, ccui, demos optional) to your CDN root (GitHub Pages repo root).
